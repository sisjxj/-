#  STM32 WiFi宠物自动喂食系统（FreeRTOS + IoT）

> 基于STM32F103C8T6的嵌入式物联网项目，实现远程控制、定时喂食与环境监测

---

## 项目简介

本项目基于STM32F103C8T6构建嵌入式控制系统，结合ESP8266实现WiFi通信，并引入FreeRTOS实现多任务调度，完成一套完整的宠物自动喂食系统。

系统支持远程控制、定时投喂、环境监测及状态反馈，具备一定工程应用价值。

---

## 系统架构
手机/上位机
↓
WiFi通信（ESP8266）
↓
STM32（FreeRTOS）
↓
├── 传感器模块（温湿度 / 重量 / 液位 / 红外）
├── 执行模块（舵机 / 继电器）
├── 显示模块（OLED）
└── 人机交互（按键 / 蜂鸣器 / 语音）

---

## ⚙️ 硬件组成

- 主控：STM32F103C8T6  
- 通信模块：ESP8266-01S  
- 传感器：
  - DHT11（温湿度）
  - HX711（重量检测）
  - 红外传感器
  - 非接触液位传感器  
- 执行器：
  - SG90舵机（投喂控制）
  - 继电器  
- 显示：OLED显示屏  
- 其他：
  - JQ6500语音模块  
  - 蜂鸣器

## 💻 软件设计

### 🔹 系统架构（FreeRTOS）

系统采用FreeRTOS进行任务划分：

| 任务 | 功能 |
|------|------|
| 传感器任务 | 采集温湿度、重量、液位数据 |
| 通信任务 | 与ESP8266通信，实现远程控制 |
| 控制任务 | 控制舵机/继电器执行喂食 |
| 显示任务 | OLED数据显示 |
| 按键任务 | 用户输入处理 |

---

### 🔹 通信设计

- 使用UART实现STM32与ESP8266通信  
- 设计简单通信协议（指令 + 数据 + 校验）  
- 支持远程控制与状态上传  

---

### 🔹 上位机

- 使用Python开发  
- 实现功能：
  - 远程喂食控制  
  - 数据显示  
  - 状态监测
